# HappyBricks V3.0 - Story Mode

## 故事驱动设计

**剧情：** 父子被落下的石块困住！用弹球打碎石块，救出他们！

## 新功能

### 营救动画（三阶段）
1. **震动阶段** - 石块清除后，屏幕震动 + 角色抖动 + 提示文字
2. **下落阶段** - 石块旋转着从顶部落下（物理模拟）
3. **庆祝阶段** - 父子跳跃庆祝 + 金色光芒 + 欢呼音效

### 角色表情系统
- **被困时**：眉头紧锁、嘴巴下弯、腮红发青、轻微抖动
- **获救后**：眯眯眼笑、嘴巴上扬、腮红粉红、跳跃庆祝

### 石块设计
- 灰色系石块（有纹理线条、高光、阴影）
- 不规则四边形形状
- 4个固定关卡布局（难度递增）

### 音效新增
- `rescue`: 大地震动 + 欢呼和声
- `shake`: 震动音效

## 技术实现
- rescuePhase 状态机控制动画
- rescueRocks 数组管理下落石头
- charState 切换表情状态
- screenShake 屏幕震动效果
