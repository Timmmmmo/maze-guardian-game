# 数值调整方案

## 📊 当前数值分析

### 问题诊断

| 问题 | 现状 | 影响 |
|------|------|------|
| 起始金币过高 | 600→350递减 | 前期太轻松，后期压力不足 |
| 塔成本偏低 | 弓箭60/炮100 | 玩家可以无脑堆塔 |
| 敌人血量偏低 | 普通60/快速35 | 塔防效果不明显 |
| 波次奖励缺失 | 无明确奖励 | 玩家没有正向反馈 |

### 数值平衡检查

```
击杀效率 = 塔DPS × 怪物停留时间 / 怪物血量

弑箭塔 vs 普通怪:
- DPS = 15 / (40/60) = 22.5
- 停留时间 ≈ 10格 / 0.50 = 20秒
- 击杀效率 = 22.5 × 20 / 60 = 7.5 (过高!)

炮塔 vs 坦克:
- DPS = 40 / (90/60) = 26.7
- 击杀效率 = 26.7 × 40 / 200 = 5.3 (仍然偏高)
```

**结论**: 当前数值偏简单，玩家几乎没有压力
