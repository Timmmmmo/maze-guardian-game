# V10 竖版射击 + 3×怪物 + 卡通拟人形象

## 目标
1. Mark V9版本 ✅
2. 玩家在屏幕下方，怪物从上到下移动 ✅
3. 怪物数量×3 ✅
4. 玩家和怪物都用拟人化卡通形象 ✅

## 布局改动
- 玩家起始位置: (bW/2, bH*0.82) — 屏幕下方中央
- 玩家活动区域: 底部45% (y > bH*0.55)
- 敌人生成位置: 屏幕顶部随机X, y = -size - random
- 敌人移动: 向下 + 轻微水平追踪玩家
- 子弹: 自动瞄准最近敌人（自然向上射击）
- 危险线: y = bH*0.52处虚线标识玩家活动边界

## 3×怪物数量
- 出怪间隔: 原1500ms ÷ 3 = ~500ms (wave 1)
- 最小间隔: 120ms (原350ms)
- 每秒约2-3个敌人，后期5-6个/秒
- 弹药适配: tile×1.0(原×0.6)，合并补弹×0.6，起始10发

## 卡通拟人形象

### 玩家 - 蓝色勇士
- 蓝色头发 + 蓝色铠甲
- 肤色圆脸 + 动漫风大眼(白底+蓝瞳+高光)
- 微笑嘴巴
- 双手持武器图标
- 轻微上下浮动动画

### Tier 0 - 绿色小鬼
- 绿色皮肤 + 绿色身体
- 斗鸡眼 + 龅牙
- 伸出的手臂

### Tier 1 - 紫色骷髅兵
- 紫色皮肤 + 小角
- 红色发光眼睛
- 横线嘴巴

### Tier 2 - 红色大恶魔
- 红色皮肤 + 弯曲大角
- 黄色发光眼 + 黑瞳孔
- 怒吼嘴巴 + 胸肌
- 更大体型

### Tier 3 - 金色BOSS
- 金色皮肤 + 皇冠(锯齿形)
- 红色瞳孔
- 更大体型 + 金色光环

## 闪避系统改动
- 冲刺改为横向闪避: 远离最近敌人的X方向
- 不再向前/向后冲，更符合竖版射击操作直觉
- 提示文字: "💨 闪避！"
