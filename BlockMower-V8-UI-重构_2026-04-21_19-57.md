# BlockMower V8 UI/UE 彻底重构

## 任务目标
修复游戏区域完全不可见的问题，按主流手游标准重构UI/UE

## 问题诊断
1. **深色背景**: 原背景 `#0a0a1e` / `#151530` 几乎全黑
2. **低对比度**: 格子背景 `#252545` 在暗底上几乎隐形
3. **方块颜色**: 使用2048经典色（米黄/浅棕）在暗底上看不见
4. **格子太小**: 44px 在深色背景上难以辨认
5. **视觉层次缺失**: 没有足够的对比度和层次感

## V8解决方案

### 1. 明亮主题（类糖果传奇/2048风格）
- 游戏背景: `#faf8ef` (经典2048米白色)
- 空格子背景: `#cdc1b4` (经典2048格子色)
- 外层容器: 深蓝渐变保持沉浸感

### 2. 高对比度方块
- 方块颜色使用2048经典色系，饱和度拉高
- 每个方块添加阴影效果
- 数字字体放大到最大28px粗体

### 3. 格子和元素放大
- 格子尺寸: 44px → 50px
- 数字字体: 16-28px（根据数字位数）
- 玩家圆圈: 18px外圈 + 12px内圈
- 敌人圆圈: 20px外圈 + 15px内圈

### 4. 角色重设计
- **玩家**: 绿色外圈 + 白色内圈 + 🎯emoji
- **敌人**: 深红外圈 + 红内圈 + 🧟emoji
- **血条**: 颜色随HP变化（绿→黄→红）

### 5. 交互优化
- 冲刺冷却: 3秒 → 2秒
- 子弹自动瞄准最近敌人
- 屏幕震动效果增强
- 粒子特效优化
