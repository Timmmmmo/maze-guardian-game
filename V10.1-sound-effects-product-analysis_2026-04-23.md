## 音效系统 (Web Audio API 合成)

使用纯代码合成音效，零外部依赖，即时可用。

### 音效列表
| 音效 | 触发时机 | 实现方式 |
|-----|---------|---------|
| shoot | 每次射击 | square波，频率800→200，根据武器调整 |
| hit | 子弹击中敌人 | sine波，400→100 |
| kill | 敌人死亡 | sawtooth波+噪声层，模拟爆炸 |
| merge | 数字合并 | sine波，音调随数值升高(300+log2(value)*100) |
| upgrade | 武器升级 | 四音符上升音阶(C5→E5→G5→C6) |
| hurt | 玩家受伤 | sawtooth波，200→80，低沉嗡声 |
| wave | 新波次 | triangle波，三音符号角声 |
| combo | 连杀 | sine波，音调随连杀数升高(600+combo*50) |
| dash | 闪避 | sine波，600→1200→400，嗖声 |

## 产品经理分析

### 当前核心循环
```
合并数字 → 获得弹药 → 自动射击 → 击杀敌人 → 得分
    ↑                                          ↓
    ←←←←←←←←←← 敌人持续涌来 ←←←←←←←←←←←←←←←←←
```

### 核心乐趣点
1. 2048合并的成就感
2. 射击的正反馈（伤害数字+粒子+音效）
3. 连杀系统的倍率奖励
4. 武器升级的期待感

### 当前问题
1. 合并和战斗割裂，需要频繁切换注意力
2. 策略深度不足，只需不断合并
3. 后期单调，缺乏变化
4. 没有持久进度

### 改进建议

#### A. 短期改进（易实现）
1. **弹药更有价值** — 高武器消耗更多弹药
2. **敌人掉落** — 击杀随机掉落弹药包/护盾/加速
3. **主动技能** — 合并特定数字解锁一次性技能
4. **连杀增益** — 连杀时武器伤害提升

#### B. 中期改进（需要设计）
1. **天赋系统** — 每局结束选择永久增益
2. **敌人多样性** — 分裂怪/护盾怪/自爆怪
3. **Boss战** — 每5波Boss，有特殊机制
4. **合并连击** — 快速连续合并有奖励

#### C. 长期改进（需要重构）
1. **肉鸽元素** — 随机武器池/敌人组合
2. **装备系统** — 合并获得装备碎片
3. **剧情/世界观**

### 最推荐的3个改动

1. **敌人掉落系统** ⭐⭐⭐⭐⭐
   - 击杀随机掉落道具
   - 道具类型：弹药包(+10)、护盾(3秒无敌)、急速(5秒×2攻速)
   - 自动飘向玩家拾取

2. **主动技能系统** ⭐⭐⭐⭐
   - 合并8 → 💣手雷投掷
   - 合并32 → ❄️冰冻光环
   - 合并128 → ⚡全屏雷暴

3. **永久天赋树** ⭐⭐⭐⭐
   - 每局结束获得天赋点
   - 天赋：初始弹药/移动速度/伤害加成/弹药效率
