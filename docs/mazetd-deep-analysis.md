## 一、项目概述

### 1.1 产品定位

```
MazeTD - 迷宫塔防游戏

核心玩法：
┌─────────────────────────────────────────────┐
│  玩家建造迷宫 + 放置防御塔 → 阻止怪物到达出口  │
```

### 2.2 核心算法分析

#### A*寻路算法

```javascript
function findPath(avoidZones=[]) {
  // BFS变体：优先选择最短路径
  
  const q = [[ENTRY.x, ENTRY.y, []]];
  const v = {};
  v[ENTRY.x+','+ENTRY.y] = 1;
  
  while(q.length) {
    const [x, y, p] = q.shift();
    
    if(x===EXIT.x && y===EXIT.y) return p.concat([[x,y]]);
    
    // 4个方向
    for(const [dx,dy] of [[0,1],[1,0],[0,-1],[-1,0]]) {
      const nx = x+dx, ny = y+dy;
      let inDanger = false;
      
      // 避开区域（侦察兵用）
      for(const zone of avoidZones) {
        if(Math.abs(nx-zone.x)<=1 && Math.abs(ny-zone.y)<=1) {
          inDanger = true; break;
        }
      }
      
      const isBlocked = grid[ny][nx] !== 0;
      
      if(!inDanger) {
        v[k] = 1;
        q.push([nx, ny, p.concat([[x,y,isBlocked]])]);
      }
    }
  }
  
  return null; // 无路径
}
```

**问题**：使用BFS而非真正的A*（无启发式函数）

